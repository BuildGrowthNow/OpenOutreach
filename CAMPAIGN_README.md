# LenGrowth LinkedIn Campaign

This directory contains the configuration for running a LinkedIn auto-pilot campaign for **LenGrowth**.

## Quick Start

1. **Create `.env`** - Copy `.env.example` and fill in your credentials
2. **Run** - Execute `docker run ...` command from `ACTIVATE.md`
3. **Onboard** - Follow the prompts to enter your product description and booking link
4. **Watch** - Monitor at http://localhost:6080/vnc.html

## What Is This?

This is **OpenOutreach** - an open-source, self-hosted LinkedIn automation tool that:
- Discovers leads using AI-generated search queries
- Qualifies profiles using Bayesian ML + LLM classification
- Sends connection requests and follows-ups
- Manages multi-turn conversations with an AI agent

## For LenGrowth

The campaign is configured to:
- Target: Founders, Growth Managers, Marketing Directors at scaling startups
- Value Prop: A comprehensive growth operating system bridging strategy and execution
- Method: AI-powered autonomous outreach with human-like conversations

## What You Need

1. A LinkedIn account
2. An OpenAI API key (or compatible)
3. Docker Desktop (recommended) or Python 3.12+
